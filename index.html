Void Tech USA - This Website

I structured multiple sections to showcase skills, projects, and contact information while implementing smooth navigation and interactive elements for an improved user experience. USB Portable Toolkit

I created an ISO using Win10XPE that contains imaging software, disk surface scanning tests, and other diagnostic programs and included this in a Ventoy boot menu with MemTest and Windows 10 and 11. Additionally, I developed several visually interactive scripts using batch and powershell scripting that includes malware removal tasks, temp file removal, restore point functionality, and more.

Active Directory Homelab with DHCP

Following Josh Makator's AD homelab guide, I designed and deployed a Windows Server 2022 domain controller in VirtualBox, integrating a Windows 10 client to simulate enterprise operations. I configured Active Directory, DNS, and DHCP services to support domain functionality, enforced security policies through Group Policy, and automated user provisioning with PowerShell. The project resulted in a fully functional home lab environment that demonstrated practical Active Directory management, user administration, and network service configuration.
